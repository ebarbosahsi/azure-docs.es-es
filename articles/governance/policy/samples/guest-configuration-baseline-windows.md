# <a name="azure-policy-guest-configuration-baseline-for-windows"></a>Base de referencia de configuración de invitado de Azure Policy para Windows

En el siguiente artículo se detallan las **máquinas Windows en \[versión preliminar\] que deben cumplir los requisitos de auditorías de la iniciativa de directiva de configuración de invitado de la base de referencia de seguridad de Azure**. Para obtener más información, consulte [Información sobre la configuración de invitado de Azure Policy](../concepts/guest-configuration.md) e [Información general sobre Azure Security Benchmark (V2)](../../../security/benchmarks/overview.md).
